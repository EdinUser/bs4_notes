Inpunotes

A small extension to make notes in localstorage from non-hidden inputs. The extension makes a JSON.stringify record. You can define a dedicated key to store notes for different pages/locations.

Get it!

Go to https://github.com/EdinUser/bs4_notes and download the content from js folder

Use it!

The simpliest way to use it just to do this

<script src="js/dynamic_notes.js" type="module"></script>
<script>
    $(function () {
        $.notesBS()
    });
</script>

Configure it

Ok, this was a simple usage with default settings. You can define more extras, using a settings object:

$.notesBS({
    key: 'test',
    useIcons: true,
    expireAfter: 168,
    iconAddNote: '<span class="material-symbols-rounded fs">note_add</span>',
    iconExistingNote: '<span class="material-symbols-rounded fs">note</span>',
    iconRemoveNote: '<span class="material-symbols-rounded fs">delete</span>',
});

All settings

List of all settings, with their default values:

const defaults = {
    //Default key name. This can later be put to URL
    key: "BS4_notes",
    //Expire time in hours
    expireAfter: 168,
    //Name of the class of the inputs
    convertClass: "haveNote",
    // Use an icon font
    useIcons: true,
    //Modal ID to handle the adding of notes
    modalId: "#addNoteModal",
    // Default Google Icon for Add a Note
    iconAddNote: '<span class="material-symbols-rounded fs">note_add</span>',
    // Default Google Icon for Existing Note
    iconExistingNote: '<span class="material-symbols-rounded fs">note</span>',
    // Default Google Icon for Remove an icon
    iconRemoveNote: '<span class="material-symbols-rounded fs">delete</span>',
    // Container for recorded notes
    recordedNotesContainer: "#notesContainer",
    // Bootstrap version
    bootstrapVersion: '4',
    // just for fun. This works for a single note, if you have multiple inputs with notes, remove this
    stickyNoteId: "",
    // You have to enter here a style for the Pushpin icon
    stickyNotePushPinIcon: '<span class="material-symbols-rounded fs">push_pin</span>',
};
